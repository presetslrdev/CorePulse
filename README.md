# CPU Monitor Notifier

Трей-приложение для Windows 11, которое следит за нагрузкой **отдельных ядер** процессора.

## Идея

Стандартные мониторы показывают общую загрузку CPU, но не замечают ситуацию, когда одно ядро
надолго «залипло» под 100% (зависший однопоточный процесс, майнер, разогнавшийся фоновый сервис) —
общая загрузка при этом может выглядеть безобидно (на 16 ядрах это всего ~6%).

CPU Monitor Notifier:

- 📊 показывает **информативную иконку в трее** — мини-диаграмма нагрузки всех ядер с цветовой
  кодировкой (зелёный / жёлтый / красный);
- 🔔 выдаёт **toast-уведомление**, если ядро (или несколько) находится под нагрузкой выше порога
  дольше заданного времени (настраивается, можно отключить);
- 🕵️ определяет и показывает **процесс-виновник**, грузящий ядро;
- ⚙️ настройки: порог %, длительность, cooldown уведомлений, автозапуск.
